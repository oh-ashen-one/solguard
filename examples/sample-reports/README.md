## Generate Your Own

```bash
# Markdown report
solguard audit ./your-program --format markdown > report.md

# JSON for programmatic access
solguard audit ./your-program --format json > report.json

# SARIF for GitHub Code Scanning
solguard ci ./your-program --sarif report.sarif
```

## Report Sections

Each report includes:

1. **Summary** — Severity counts and pass/fail verdict
2. **Critical Findings** — Must fix before deployment
3. **High Severity** — Significant vulnerabilities
4. **Medium Severity** — Potential issues
5. **Low Severity** — Best practice recommendations
6. **Scan Details** — Patterns checked, time, files scanned
7. **Certificate** — For passing audits
